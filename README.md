# 한국사 암기 퀴즈

설명(문제) → 답안보기 → 다음(랜덤) 형태의 아주 가벼운 암기 퀴즈입니다.

## 실행

- `index.html`을 브라우저로 열면 됩니다.
- (권장) VS Code/Cursor의 Live Server 같은 간단한 정적 서버로 열면 더 안정적입니다.

## 사용법

- **답안보기**: 답(인물)이 표시됩니다. 다시 누르면 숨깁니다.
- **다음 →**: 다음 문제가 랜덤 순서로 나옵니다. 한 바퀴를 돌면 다시 랜덤으로 섞입니다.
- **섞기**: 즉시 랜덤 순서를 새로 만듭니다.

## 데이터 추가/수정

화면 아래 **문제/답 데이터 편집하기**를 열고, JSON 배열을 수정한 뒤 **적용**을 누르세요.

형식:

```json
[
  { "answer": "여운형", "prompt": "조선 건국 동맹을 결성하였다." },
  { "answer": "여운형", "prompt": "좌우합작운동을 추진하였다." }
]
```
